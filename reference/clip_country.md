# Function to clip national maps from global WorldPop sex and age disaggregated population maps

Function to clip national maps from global WorldPop sex and age
disaggregated population maps

## Usage

``` r
clip_country(r, adm)
```

## Arguments

- r:

  Object of class SpatRaster, Raster or their path

- adm:

  object of class sf with the location of subnational units
