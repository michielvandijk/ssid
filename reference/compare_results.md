## Usage

``` r
compare_results(seed, targets)

compare_results(seed, targets)
```

## Arguments

- seed:

  `data.frame` Seed table with a weight column in the same format
  required by
  [`ipu()`](https://michielvandijk.github.io/ssid/reference/ipu.md).

- targets:

  `named list` of `data.frames` in the same format required by
  [`ipu()`](https://michielvandijk.github.io/ssid/reference/ipu.md).

## Value

`data frame` comparing balanced results to targets
