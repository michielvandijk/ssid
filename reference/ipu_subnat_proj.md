# ipu_subnat_proj

Function to rake subnational population projections so that they match
national age-sex and subnational pop projections.

## Usage

``` r
ipu_subnat_proj(y, scen, verbose = TRUE)
```

## Arguments

- y:

  Year

- scen:

  Scenario

- verbose:

  Print iteration details and worst marginal stats upon completion
  generated by
  [`ipfr::ipu_matrix`](https://rdrr.io/pkg/ipfr/man/ipu_matrix.html)?
  Default TRUE.

- param:

  ssid_par
