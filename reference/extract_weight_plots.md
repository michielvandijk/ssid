# Extract figures with with weight distribution

Extract figures with with weight distribution

## Usage

``` r
extract_weight_plots(sim, ssp_y, reg_tz)
```

## Arguments

- sim:

  Object with spatial simulation results.

- ssp_y:

  Vector with ssp-year combinations for which simulation results are
  stored in `sim`.

- reg_tz:

  Vector with regions for which simulation results are stored in `sim`.
